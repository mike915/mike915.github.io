--- Mar 29 to April 4 ---

Set Up Development Environment

  1. Xcode:
    • The latest Xcode is already in my Mac Pro.
    • Learned how to develop a simple iOS app following the Udacity course Intro to iOS App Development with Swift in last Xmas holiday.
  2. Android Studio:
    • Just installed on my Mac.
    • Android 5.1 (API 22) + Android Support Library

Go Through Parse Document "iOS Guide"

  • Focus on "Objects", "Queries" and "Users".
  • "User Interface" is pretty convenient.
  • Dashboard->Data->Data browser: directly edit, add, delete data.

Go Through "Anypic" Tutorial

  • Focus on "Data Model" Chapter
  • Data model == Classes ?
  • Need to figure out how/where to create a new data object.

Others

Some Internal Test Tools
  1. TestFlight:
    • Supported by Apply,
    • Merely support for iOS apps.
  2. 内测宝:
    • Support for both iOS and Android.
Register Formal email, Wechat, Facebook, Weibo, etc.
